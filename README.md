# Commandes utiles

Get une nouvelle bibliotheque :
```go 
go get 
``` 

Run le back : 
```go
go run . 
``` 

Permet de creer un module : 
```go
go mod init 
``` 

Permet de telecharger ce qu'il y a dans le module :
```go
go mod  
```
